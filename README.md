# Ghost Market - Turn Based RPG Prototype
made with **Unity 2022.3** for a Programmer Pre-Interview Test.

## Story

Player terjebak di **Pasar Hantu** dan harus mencari **Ghost Key** yang disembunyikan oleh salah satu pedagang hantu. Setelah mendapatkan kunci, pemain dapat menghadapi iblis penjaga untuk keluar dari pasar.

## Features

- World exploration (WASD)
- NPC Interaction
- Dialogue System (Scriptable Object)
- Simple Cutscene (Auto Walk)
- Turn Based Battle
- Easy Enemy AI (Attack / Defend / Heal)
- Scene Transition (Fade In / Fade Out)

## Controls

- W --> Move Forward
- A -- > Move Left 
- S -- > Move Backward 
- D -- > Move Right 
- Space -- > Interact 
- Mouse -- > Select Battle Action 

## How to Play

1. Explore the Ghost Market.
2. Talk to the ghost merchants.
3. Find the Ghost Key.
4. Defeat the Demon in a turn-based battle.
5. Escape the Ghost Market.
